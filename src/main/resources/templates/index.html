Solar System weather report

This application is an excercise to calculate the weather in a solar system basen on a group of rules.

The application exposes two different endpoints.

.
Available endpoints:
https://solarsystem-230621.appspot.com/{day} This get the weather for a especific day
https://solarsystem-230621.appspot.com/report/{from day}/{to day}] This get a report for a group of days.

En caso de que los dias solicitados no se hayan calculado todavia, la aplicacion respondera un mensaje diciendo que el reporte esta incompleto y procedera a calcularlo en el background.
Una vez este completo, se podra volver a solicitar el reporte y se recibiran todos los datos.
